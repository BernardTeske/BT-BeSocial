BT-BeSocial
===========

This is a Font for Social Icons. At the Moment it only contains some Characters.

##Characters

* F for Facebook-Icon
* T for Twitter-Icon
* Y For Youtube-Icon
* R for RSS-Icon
* X for XING-Icon

Feel free to use!

View characters here: https://github.com/BernardTeske/BT-BeSocial/blob/master/view.jpg
